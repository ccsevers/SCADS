10 minute stats

Rain Stats

Rain is a workload generator. We display the average request latency as measured at the Rain node for each of the apps.

Web Tier Stats

These graphs report the aggregate request rate measured at each app's web servers. The web app director scales the number of web servers dedicated to each app in response to the average CPU utilization.


PIQL Stats

PIQL is a library running on the app servers. Each app server reports its 99th percentile query latency.

SCADS Stats

SCADS is an incrementally scalable key-value store. Each namespace has its own set of SCADS nodes. The SCADS director scales each namespace's SCADS cluster in response to the workload of the storage nodes.


Mesos Stats

Mesos allows multiple frameworks to share a cluster. Interactive and analytics workloads can coexist on our cluster, which changes size over time.
